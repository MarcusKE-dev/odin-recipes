# Odin Recipes

## Project Description
This project is part of The Odin Project curriculum. It is a simple recipe website built to demonstrate foundational web development skills.

## Skills Demonstrated
In this project, I have successfully demonstrated the following technical skills:

*   **HTML Structure & Organization**: Using semantic tags like `<h1>`, `<h2>`, and lists (`<ul>`, `<ol>`) to organize recipe details logically.
*   **File Management**: Creating project directories and managing files using the terminal.
*   **Relative Linking**: Navigating between a root `index.html` file and sub-pages located within a `recipes/` directory.
*   **Asset Management**: Implementing images using relative file paths and descriptive `alt` text for accessibility.
*   **Boilerplate implementation**: Setting up proper HTML5 document structures from scratch.

## Project Structure
- `index.html` (Main landing page)
- `recipes/` (Directory containing individual recipe pages)
- `images/` (Directory for project image assets)
